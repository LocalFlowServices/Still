# Still

Cross-loader Minecraft mod bootstrap targeting **Fabric**, **Forge**, and **NeoForge**.

## Quick start

1. Install JDK 21.
2. Update versions in `gradle.properties` as needed.
3. Run Gradle tasks for the loader you want to work on.

## Modules

- `common` – shared mod logic
- `fabric` – Fabric entrypoint
- `forge` – Forge entrypoint
- `neoforge` – NeoForge entrypoint
